## 7. 安全与依赖基线

| 类别 | 最低要求 | 验证方式 |
| ---- | -------- | -------- |
| 依赖安全 | 依赖安装后须可运行 `npm audit` / `pip audit` / 等价命令（按栈选用） | QA 记录命令、退出码、摘要 |
| 密钥与配置 | 禁止硬编码密钥；敏感配置走环境变量 | secrets 扫描或人工检查 |
| 输入校验 | 外部输入须有校验与错误处理 | 单元/集成测试覆盖 |
| 许可证 / 供应链 | 新增依赖须无明显许可证与供应链风险 | 依赖清单人工核查或工具输出 |
